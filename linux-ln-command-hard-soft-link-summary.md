ln命令
硬链接
创建硬链接的语法: ln 源文件名 硬链接文件名 
硬链接就是同一个文件取多个别名
给一个文件取别名，修改一个文件的内容，相对应的别名的文件内容也会被修改
link file1 file1.hard
原理
体现Linux管理文件的方式---数字化管理
文件的本质时innode编码，每次创建一个硬链接，innode引用计数对应增加1
每次删除一个同编码文件时，引用计数减1，当引用计数减小到0的时候，才是真正的删除了对应的文件
查看文件编码的命令stat 文件名
不能给目录创建硬链接

软连接
语法ln -s 源文件名 软连接文件名
ln -s file2 file2.soft  
软连接是一个文件，具有独立的incode，查看软连接时查看软连接连接文件的地址
所以创建软连接时建议使用绝对路径而不是相对路径，因为一旦搬移软连接，他存的相对路径不会随机应变就会失效
相当于windows下的快捷方式
