ls命令

\-l 显示详细信息

详细信息有7个部分

drwxr-xr-x 3 tarena tarena     4096 5月  18  2020 5.4

1.前10个字符

1）第一个字符，文件类型

2）后9个字符，文件的访问属性

2.硬链接计数

3.文件所有者

4.文件所属组

5.文件大小

对于普通文件，直接存储实际大小

对于目录文件，是目录所占用磁盘的空间大小。---4k的整数倍

6.文件创建/修改时间

7.文件名

\-a：all，显示所有的文件，包括隐藏文件

linux系统中，隐藏文件使用“.”字符开头的文件，为隐藏文件

&#x20;-R：递归显示目录中的内容

可以将多个参数组合使用。ls -R -a-l或者 ls-alR 参数的顺序无所谓，但是参数必须跟在命令后面

ls -h:显示其他的空间大小单位，默认的单位是字节

不常用的：

\-F：显示时，添加文件提示符，表示文件类型

\-d：显示目录文件本身属性

默认，对于目录而言，显示的是目录本身的字内容的详细信息

文件类型

Lixun操作系统中一共有7种类型

1.普通文件，用“-”表示

2.目录文件：用“d”表示

3.软连接文件：用“小L”表示，相当于windows里面的“快捷方式”

4.管道文件：使用“p”表示，pipe，进程间的快速传输

5.套接字文件：用“s”表示，socket，两个文件专用的“通信通道文件”

6.字符设备文件：使用“c”表示，character，像流水线一样一个接着一个的传，不能随机不能跳

7.块设备文件：使用“b”表示，block，相当于windows里面的磁盘

后四种文件是伪文件，不占用磁盘存储

前三种文件是真正的文件，消耗磁盘存储

文件访问属性

一共有9个字符组成，分为三组，每组rwx，没有对应权限则使用-代替

三组依次是：所有者，所属组，其他人

所有者：创建该文件的用户

所属组，创建用户默认产生用户组，并自动将用户添加到用户组

其他人：既不是所有者，又不是所属组就是其他人

就是权限分配，所属的人有什么，他的组里其他人有什么，其他人有什么权限

r：读权限

w：写权限

x：执行权限
