其他文件操作命令
创建空文件
touch+文件名
创建空目录
mkdir +目录名
linux系统中不以后缀名作为区分文件的依据
linux认不出来后缀是个啥，只当个文件，加后缀只是方便自己看而已

一次创建多级目录使用-p选项
举例：mkdir a/b/c/d -p
mkdir -p a/b/c/d这个位置顺序没什么影响
which
查看命令所对应的可执行文件所在目录位置
举例：which data，他就能显示具体所在位置

rmdir
用来删除空目录，目录非空不能删除
rmdir d ，d必须为空
rm命令删除的文件不能够恢复，只能做磁盘回复
rm常用参数
-r递归：删除多级子目录中的所有内容
-i：与用户交互式删除，让用户进行删除操作时进行二次确认
回答y/n

\-f：强制删除
rm 文件名和rm -f 文件名 效果是一样的
建议写的使用把-f 和 -i写上，避免误删

cp
语法： cp 参数A 参数B
参数A是文件
参数B是目录，且目录存在，通过cp将文件拷贝到目录当中
参数B不存在，且不是目录，将文件A拷贝，生成B bbb就是简单的文件
参数B不存在，且是目录，报错 bbb/就是目录
参数B是文件，则A覆盖B
参数A是目录

//  a/这里的/是目录提示符

要拷贝的话必须加上 -r或者-a

\-r 递归 相当于新建文件

\-a All 拷贝时带有文件默认属性，包括文件创建的时间。。。



mv命令

mv操作文件和操作目录都不需要额外参数

作用1：改名

mv 源文件 目标文件。目标文件不存在则直接改名

作用2：移动

mv 源文件 目标目录。目标目录存在，移动



cat命令

cat文件名，用于查看文件中的信息

cat file1 > file2

file 2不存在则新建，把file1的内容写入

file2如果存在，则file1的内容将file2覆盖



cat file1>>file2

如果file2不存在，新建，将file1的内容写入

file2如果存在，则将file1的内容追加到file2中

tac把cat倒着写，就把文件倒叙展示
